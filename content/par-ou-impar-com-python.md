Como vocês viu gosto muito de matemática, então nada melhor de juntar
Matemática com Python, um script simples em Python para identificar se o
numero digitado é Par ou Ímpar.  

~~~~ {.brush:python}
#!/usr/bin/env python
n = raw_input("Numero: ")
if n%2==0:
    print "Par"
else:
    print "Impar"
~~~~
